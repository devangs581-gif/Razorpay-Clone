# 💳 Razorpay Clone

A modern and responsive clone of the Razorpay website, built to replicate the UI/UX and core design of a real-world payment gateway platform.

---

## 🚀 Features

* 💻 Fully responsive design (mobile + desktop)
* 🎨 Clean and modern UI inspired by Razorpay
* 🧾 Payment section layout simulation
* 🔐 Secure-looking interface (UI only, no real transactions)
* ⚡ Smooth navigation and interactive elements

---

## 🛠️ Technologies Used

* HTML
* CSS
